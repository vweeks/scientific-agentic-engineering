# Introduction to Scientific Agentic Engineering

*This content is under development as part of Milestone 1 (July 2026).*

Scientific Agentic Engineering is a methodology for integrating AI coding agents into scientific software development workflows. It addresses the gap between the rapid adoption of AI tools and the principled engineering practices that scientific software demands.
